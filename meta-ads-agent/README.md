# Meta Ads Optimizer

Meta Ads Performance Analyse & Optimierung

## Quick Start

```bash
# Agent starten
openclaw agent --agent meta-ads -m "[deine Anfrage]"
```

## Was der Agent kann

- 📊 **KPI-Analyse** — CTR, CPC, CPM, ROAS, CPA interpretieren
- 🔍 **Performance-Diagnose** — Warum laufen Ads so wie sie laufen?
- 📈 **Benchmark-Vergleiche** — Wie gut sind deine Zahlen wirklich?
- 💡 **Optimierungs-Tipps** — Konkrete Handlungsempfehlungen

## Daten übergeben

- Screenshots vom Ads Manager
- CSV/Excel Exports
- PDF Reports
- Einfach beschreiben was du siehst

## Beispiele

```bash
# Screenshot analysieren lassen
openclaw agent --agent meta-ads -m "Hier ist ein Screenshot meiner aktuellen Kampagne. Die CTR ist bei 0.8% — ist das gut?"

# Benchmarks abfragen
openclaw agent --agent meta-ads -m "Was sind aktuelle Benchmarks für CPC im Fashion-Bereich?"

# Performance-Probleme diagnostizieren
openclaw agent --agent meta-ads -m "Meine Kampagne hat plötzlich 40% weniger Reach. Woran könnte das liegen?"
```
