# 📘 Google Tag Manager — Zugang einrichten

## Zugang gewähren

1. Gehe zu **tagmanager.google.com**
2. Wähle deinen Container aus
3. **Verwaltung** (oben) → **Nutzerverwaltung**
4. **+** → **Nutzer hinzufügen**
5. E-Mail: **deniz@adsdrop.de**
6. Berechtigung: **Veröffentlichen** (damit wir Tags live schalten können)
7. **Einladen**

## Falls noch kein GTM vorhanden

Kein Problem — wir richten den GTM für dich ein:

1. Wir erstellen einen Container in unserem GTM-Konto
2. Du bekommst einen Code-Schnipsel für deinen Shop
3. Bei Shopify: Einfach über die Theme-Einstellungen oder ein GTM-Plugin

## Was wir von dir brauchen:

| Info | Beispiel |
|------|---------|
| GTM Container ID | GTM-XXXXXXX |

💡 **Tipp:** Die Container ID findest du oben in der Mitte im Tag Manager Dashboard (Format: GTM-XXXXXXX). Falls du noch keinen hast, sag uns Bescheid!
