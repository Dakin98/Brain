# Shopify Access Token erstellen (Onboarding-Call)

## Im Shopify Admin des Kunden:

1. **Settings** → **Apps and sales channels** → **Develop apps**
2. Falls nötig: "Allow custom app development" aktivieren
3. **Create an app** → Name: `adsdrop Reporting`
4. **Configuration** → **Admin API access scopes** → `read_orders` auswählen → Save
5. **Install app** → Token kopieren

## Danach:

Sag mir den Token + Shop-URL, ich trag beides in Airtable ein.

Oder manuell in Airtable → Kunden → Kunde auswählen:
- **Shopify Shop URL**: `https://SHOPNAME.myshopify.com`
- **Shopify Access Token**: `shpat_xxxxx...`

⚠️ Token wird nur EINMAL angezeigt — sofort kopieren!
