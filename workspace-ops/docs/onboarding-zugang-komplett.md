# 🚀 adsdrop Onboarding — Zugänge einrichten

Hey! Damit wir direkt für dich loslegen können, brauchen wir Zugang zu ein paar Plattformen. Keine Sorge — dauert nur wenige Minuten pro Plattform.

---

## 1️⃣ Meta (Facebook) Ads

**Zugang geben:**
1. Gehe zu **business.facebook.com** → Einstellungen → **Partner** → **Hinzufügen**
2. Unsere Business Manager ID: **[ADSDROP_BM_ID]**
3. Berechtigungen: Werbekonten + Pixel verwalten ✅

**Was wir brauchen:**
- Business Manager ID (findest du unter Einstellungen → Unternehmensinformationen)
- Ad Account ID (Format: act_123456789)
- Pixel ID (findest du im Events Manager)

---

## 2️⃣ Google Ads

**Zugang geben:**
1. Gehe zu **ads.google.com** → Tools → **Zugriff und Sicherheit**
2. Nutzer hinzufügen: **deniz@adsdrop.de**
3. Zugriffsebene: Standard oder Admin

**Was wir brauchen:**
- Kundennummer / CID (Format: 123-456-7890, oben rechts in Google Ads)

---

## 3️⃣ Google Analytics 4

**Zugang geben:**
1. Gehe zu **analytics.google.com** → Verwaltung (⚙️)
2. Property-Zugriffssteuerung → Nutzer hinzufügen: **deniz@adsdrop.de**
3. Rolle: Bearbeiter

**Was wir brauchen:**
- GA4 Property ID (unter Verwaltung → Property-Einstellungen)

---

## 4️⃣ Google Tag Manager

**Zugang geben:**
1. Gehe zu **tagmanager.google.com** → Verwaltung → Nutzerverwaltung
2. Nutzer hinzufügen: **deniz@adsdrop.de**
3. Berechtigung: Veröffentlichen

**Was wir brauchen:**
- Container ID (Format: GTM-XXXXXXX)

Falls du noch keinen GTM hast — kein Problem, wir richten das für dich ein!

---

## 5️⃣ Shopify (falls zutreffend)

Wird gemeinsam im Onboarding-Call eingerichtet. Dauert 2 Minuten.

---

## 6️⃣ Klaviyo (falls zutreffend)

**Was wir brauchen:**
- Private API Key (Klaviyo → Settings → API Keys → Create Private Key → Full Access)

---
