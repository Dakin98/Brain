# Quick-Start Guide: adsdrop ClickUp System

## Übersicht

```
🟢 Growth          → CRM & Marketing (Sales Pipeline, Content, Outbound)
🔵 Delivery        → Kunden-Projekte (1 Folder pro Kunde)
🟣 Creative Ops    → Creative Testing System (Pipeline, Ideas, Learnings)
⚪ Operations      → Interne Projekte & Admin
```

---

## Täglicher Workflow

### Morgens (5 Min)
1. Öffne **Creative Operations > 🎯 Creative Pipeline** (Board View)
2. Check: Welche Tasks sind in "Review" oder "Adjustments"?
3. Check: Was ist heute fällig? (Calendar View)

### Bei neuer Creative-Idee
1. Gehe zu **Creative Ops > 🎯 Creative Pipeline**
2. Erstelle neuen Task im Status **"idea"**
3. Fülle Custom Fields aus:
   - **Client** (Pflicht!)
   - **Concept Type** (Video/Static/Motion)
   - **Creative Type** (Net New oder Iteration)
   - **Hook Type** (welcher Hook wird getestet?)
   - **Ad Format** (UGC, Testimonial, etc.)
4. Beschreibe das Konzept in der Task Description
5. Wenn Briefing fertig → Status auf **"briefing"**

### Creative durch die Pipeline bewegen
```
idea → briefing → content → cutting → review → adjustments → r2l → launched
```

- **idea:** Rohe Idee, noch nicht ausgearbeitet
- **briefing:** Briefing wird geschrieben / Creator wird gebrieft
- **content:** Filming/Content-Erstellung läuft
- **cutting:** Editor schneidet/bearbeitet
- **review:** Internes Review durch Strategist/Lead
- **adjustments:** Änderungen nach Review
- **r2l (Ready to Launch):** Freigegeben, wartet auf Launch
- **launched:** Live in Ad Account, wird getestet

### Nach dem Launch
1. Warte 5-7 Tage für Ergebnisse
2. Trage **Performance Score (ROAS)** ein
3. Setze **Result** auf Winner 🏆, Loser ❌, oder Inconclusive 🤷
4. Bei Winner: Erstelle Iteration-Tasks (Creative Type = "Iteration")
5. Dokumentiere Learnings in **📝 Creative Learnings**

---

## Neuer Kunde onboarden

1. CRM: Task auf **"Won 🏆"** setzen
2. **Delivery > Clients:** Neuen Folder mit Kundenname erstellen
3. Onboarding-Template anwenden:
   - Gehe zu **📋 _Onboarding Template**
   - List Menu → "Save as Template" (einmalig)
   - Neue Liste im Kunden-Folder → "From Template" → Onboarding Template wählen
4. Checklisten durcharbeiten (Vertrag → Account → Creative → Tooling → Kickoff)
5. Client im **Creative Pipeline > Client Dropdown** hinzufügen

## Troubleshooting

| Problem | Lösung |
|---|---|
| Custom Field fehlt | Creative Pipeline > Settings > Custom Fields > "Add existing field" |
| Falscher Status | Sicherstellen, dass du in der richtigen Liste bist (Creative Pipeline hat eigene Statuses) |
| Kann Client nicht auswählen | Client muss erst im Dropdown hinzugefügt werden (List Settings > Custom Fields > Client > Edit) |
| Task verschwunden | Prüfe: Wurde Filter gesetzt? Ist der Task vielleicht in "launched" (geschlossen)? |
| Duplicate Custom Fields | In List Settings > Custom Fields die Duplikate (Space-Level) entfernen |
