# Growth System für adsdrop

## Content Engine + Outbound Engine = Full Growth Stack

**Strategie:** 
- **Inbound:** YouTube Long-Form als Hub → Reels + LinkedIn + Newsletter
- **Outbound:** Cold Mailing für eigene Lead-Generierung
- **Ziel:** 1x/Woche Content + kontinuierliche Outbound-Kampagnen

**Status:** Konzept-Phase

---

## 🎯 CONTENT STRATEGIE

### Der Flywheel

```
YouTube Long-Form Video (1x/Woche)
    ↓
├── 🎬 Reels/TikTok Shorts (3-5x/Woche)
├── 💼 LinkedIn Posts (2-3x/Woche)  
├── 📧 Newsletter (1x/Woche)
└── 📝 Blog/SEO (optional)
```

### Content Pillar (Themen)

| Pillar | YouTube | LinkedIn | Reels |
|--------|---------|----------|-------|

| **Case Studies** | Kundenstories | Results | Behind the Scenes |
| **Agentur Life** | Vlogs | Learnings | Funny moments |
| **Industry Trends** | Analysen | Commentary | Hot takes |

### 1. Content Ideas (Backlog)

**Zweck:** Sammeln & Priorisieren aller Content-Ideen

**Custom Fields:**
- Pillar (Meta Ads, Case Study, Agentur Life, Trends)
- Format (YouTube, LinkedIn, Reels, Newsletter)
- Priority (S, A, B, C)
- Status (Idea, Approved, In Production, Done)

- Effort (Low, Medium, High)

**Views:**
- Board nach Pillar
- Table nach Priority
- Calendar (Publish-Dates)

### 2. YouTube Pipeline

**Workflow (Status):**
```
Idea → Script/Outline → Recording → Editing → Thumbnail → SEO → Review → Scheduled → Published → Distributed
```

**Tasks pro Video:**
1. **Script/Outline** (Checklist)
   - [ ] Hook (erste 30 Sek)
   - [ ] Main Content strukturiert
   - [ ] CTA (Subscribe, Comment, etc.)
   - [ ] B-Roll Plan

2. **Recording** (Checklist)
   - [ ] Setup Licht/Kamera
   - [ ] Audio-Check
   - [ ] Main Take
   - [ ] B-Roll Shots
   - [ ] Backup auf NAS

3. **Editing** (Checklist)
   - [ ] Rough Cut
   - [ ] B-Roll einfügen
   - [ ] Grafiken/Animationen
   - [ ] Color Grade
   - [ ] Sound Design
   - [ ] Final Export (4K)

4. **Thumbnails** (Checklist)
   - [ ] 3 Varianten designen
   - [ ] A/B Test Setup
   - [ ] Final auswählen

5. **SEO & Upload** (Checklist)
   - [ ] Title (SEO + Clickbait)
   - [ ] Description mit Timestamps
   - [ ] Tags recherchieren
   - [ ] Endscreens einfügen
   - [ ] Cards setzen
   - [ ] Scheduled Publish

6. **Distribution** (Checklist)
   - [ ] Community Post
   - [ ] Newsletter versenden
   - [ ] LinkedIn Post
   - [ ] Stories/Reels Teaser

**Custom Fields:**
- Video Title
- Publish Date
- Target Audience
- Keywords
- Thumbnail Variants
- Video URL (nach Publish)
- Performance (nach 7 Tagen)

### 3. Reels Pipeline

**Workflow:**
```
YouTube Video Published → Extract Hooks → Edit Reels → Schedule → Publish
```

**Automation:** Wenn YouTube Video auf "Published" → automatisch 3 Reels-Tasks erstellen

**Tasks pro Reel:**
1. **Hook Extraction**
   - [ ] Beste Momente identifizieren (3-5)
   - [ ] Timestamps notieren
   - [ ] Hook-Text schreiben

2. **Editing**
   - [ ] Raw Cut (9:16)
   - [ ] Captions/Subtitles
   - [ ] Hook-Overlay (erste 3 Sek)
   - [ ] Music/Sound
   - [ ] Export

3. **Publishing**
   - [ ] Description mit CTA
   - [ ] Hashtags
   - [ ] Schedule (beste Zeit)
   - [ ] Cross-Post (Reels + TikTok + YT Shorts)

**Custom Fields:**
- Source Video (Link zu YouTube Task)
- Hook Timestamp
- Platform (Reels, TikTok, YT Shorts)
- Publish Date
- Views (nach 7 Tagen)

### 4. LinkedIn Pipeline

**Workflow:**
```
YouTube Video Published → Write Post → Carousel/Video → Publish → Engage
```

**Content-Typen:**
- **Long-Form Text:** Key Learnings aus Video
- **Carousel:** 5-10 Slides mit Insights
- **Video:** 2-3 Min Snippet
- **Poll:** Engagement bait

**Tasks pro Post:**
1. **Write Post**
   - [ ] Hook (erste 2 Zeilen)
   - [ ] Story/Value
   - [ ] CTA
   - [ ] Format wählen (Text/Carousel/Video)

2. **Create Asset**
   - [ ] Carousel designen (Canva)
   - [ ] ODER: Video schneiden
   - [ ] Preview check

3. **Publish & Engage**
   - [ ] Posten (beste Zeit: 8-9 Uhr)
   - [ ] First Comment (Mehrwert)
   - [ ] 30 Min Engagement (Antworten)
   - [ ] 24h später: weitere Kommentare

**Custom Fields:**
- Post Type (Text, Carousel, Video, Poll)
- Source Video (Link)
- Publish Date
- Engagement Rate (nach 48h)

### 5. Newsletter Pipeline

**Workflow:**
```
YouTube Video Published → Write Newsletter → Design → Send → Archive
```

**Zwei Newsletter-Typen:**
1. **Weekly Digest:** Video + LinkedIn Highlights + Agency Updates
2. **Deep-Dive:** Einzelnes Video als ausführlicher Artikel

**Tasks:**
1. **Write**
   - [ ] Subject Line (3 Varianten)
   - [ ] Intro
   - [ ] Video Summary
   - [ ] LinkedIn Best-of
   - [ ] CTA

2. **Design**
   - [ ] Klaviyo Template
   - [ ] Thumbnail einfügen
   - [ ] Links checken
   - [ ] Mobile Preview

3. **Send & Analyze**
   - [ ] A/B Test Subject Line
   - [ ] Schedule (beste Zeit)
   - [ ] 24h Stats checken
   - [ ] Archive

**Custom Fields:**
- Newsletter Type (Digest, Deep-Dive)
- Subject Line
- Open Rate
- Click Rate
- Unsubscribes

### 6. Distribution Checklist (Master)

**Zweck:** Sicherstellen, dass jeder Content überall landet

**Checkliste pro YouTube Video:**
- [ ] YouTube Published

- [ ] Reels #3 (3rd Hook)

## 📧 2. OUTBOUND ENGINE (Cold Mailing)

### Strategie: Eigene Lead-Generierung für adsdrop

**Ziel:** 50-100 neue Prospects/Woche → 5-10 Quali Calls/Monat  
**Zielgruppe:** E-Commerce Brands (DACH) mit 500k-5M Umsatz  
**Tool:** Instantly (Domain-Rotation, Inbox-Warming)

### ClickUp Struktur

```
Growth Space
└── 📁 Marketing
    ├── 📁 Content Engine (siehe oben)
    └── 📁 Outbound Engine (NEU)
        ├── 📋 Domain & Inbox Management
        ├── 📋 Lead Lists (ICP Segmente)
        ├── 📋 Campaigns (laufende Kampagnen)
        ├── 📋 Sequences (Email Templates)
        ├── 📋 Reply Management
        └── 📋 Outbound Analytics
```

### 2.1 Domain & Inbox Management

**Zweck:** Überblick über alle Domains und Email-Accounts

**Workflow:**
```
Domain kaufen → DNS Setup → Inbox erstellen → Warming starten → Aktiv
```

**Tasks pro Domain:**

1. **Domain Setup**
   - [ ] Domain kaufen (z.B. ads-drop.de, getadsdrop.com)
   - [ ] DNS: MX Records (Google Workspace/Zoho)
   - [ ] DNS: SPF Record
   - [ ] DNS: DKIM Record
   - [ ] DNS: DMARC Record
   - [ ] DNS: Custom Tracking Domain (für Instantly)

2. **Inbox Setup**
   - [ ] 2-3 Email Accounts erstellen
   - [ ] Profile vervollständigen (Name, Foto, Signatur)
   - [ ] In Instantly hinzufügen
   - [ ] Warming starten (2-4 Wochen)

3. **Monitoring**
   - [ ] Daily: Inbox Health check
   - [ ] Weekly: Deliverability Score
   - [ ] Monthly: Domain Rotation Plan

**Custom Fields:**
- Domain Name
- Provider (Google/Zoho)
- Anzahl Inboxen
- Warming Status (Day X/Y)
- Health Score (%)
- Status (Warming | Ready | Active | Pause)

### 2.2 Lead Lists (ICP Segmente)

**Zielgruppen (ICPs):**

| ICP | Kriterien | Quelle |
|-----|-----------|--------|
| **Fashion DACH** | Shopify, DE/AT/CH, 500k-2M Umsatz | Apollo, LinkedIn |
| **Beauty DACH** | Shopify, DE/AT/CH, 300k-1M Umsatz | Apollo, LinkedIn |
| **Home & Living** | Shopify, DE/AT/CH, 500k-5M Umsatz | Apollo, LinkedIn |
| **Food & Bev** | Shopify, DACH, 1M+ Umsatz | Apollo, LinkedIn |

**Workflow:**
```
ICP definieren → Leads suchen → Liste aufbereiten → Validieren → Hochladen
```

**Tasks pro Liste:**

1. **Research**
   - [ ] Apollo Suche (Filter: ICP)
   - [ ] LinkedIn Sales Navigator
   - [ ] Website-Check (Shopify erkennen)
   - [ ] Ergebnis: 200-500 Leads

2. **Cleaning**
   - [ ] Duplikate entfernen
   - [ ] Unvollständige Daten löschen
   - [ ] Enrich (Name, Title, Company Size)
   - [ ] Validieren (Email-Verifier)

3. **Upload & Segment**
   - [ ] In Instantly hochladen
   - [ ] Tags vergeben (ICP, Größe, etc.)
   - [ ] Campaign zuordnen

**Custom Fields:**
- ICP Segment
- Lead Count
- Quelle (Apollo, LinkedIn, etc.)
- Upload Date
- Campaign Status

### 2.3 Campaigns

**Campaign-Struktur:**

```
Campaign: [ICP] + [Angle] + [Monat]
Beispiel: "Fashion_Q1_Angle_CaseStudy"
```

**Workflow:**
```
Angle wählen → Sequence schreiben → Leads zuordnen → Launch → Monitor → Optimize
```

**Tasks pro Campaign:**

1. **Campaign Setup**
   - [ ] Angle definieren (Problem, Solution, Case Study)
   - [ ] Sequence auswählen/erstellen
   - [ ] Lead List zuordnen
   - [ ] Sende-Limit festlegen (20-40/Tag pro Inbox)
   - [ ] Schedule (Tage/Zeiten)

2. **Launch**
   - [ ] Test-Emails senden
   - [ ] Campaign aktivieren
   - [ ] Daily Monitoring einrichten

3. **Optimization**
   - [ ] Daily: Reply Rate checken
   - [ ] Weekly: A/B Test Ergebnisse
   - [ ] Bi-weekly: Sequence anpassen
   - [ ] Monthly: Neue Angle testen

**Custom Fields:**
- Campaign Name
- ICP Segment
- Sequence
- Lead Count
- Status (Draft | Active | Paused | Completed)
- Launch Date
- Reply Rate (%)
- Meeting Rate (%)

### 2.4 Sequences (Email Templates)

**Sequence-Typen:**

| Sequence | Steps | Use Case |
|----------|-------|----------|
| **Cold Intro** | 5-7 | Erster Kontakt |
| **Case Study** | 4-6 | Social Proof Angle |
| **Problem-Agitate** | 5-7 | Pain Point Angle |
| **Breakup** | 3 | Letzter Versuch |

**Template-Struktur:**

1. **Subject Line Variants**
   - A: Question
   - B: Value Prop
   - C: Curiosity

2. **Email Body**
   - Personalization ({{first_name}}, {{company}})
   - Hook (erste Zeile)
   - Value/Problem
   - CTA
   - PS

3. **Follow-ups**
   - Day 3: Bump
   - Day 7: Value Add
   - Day 12: Case Study
   - Day 18: Breakup

**Custom Fields:**
- Sequence Name
- Steps Count
- Open Rate (%)
- Reply Rate (%)
- Status (Testing | Winner | Archive)

### 2.5 Reply Management

**Zweck:** Alle Antworten strukturiert bearbeiten

**Reply-Kategorien:**

| Kategorie | Action | Status |
|-----------|--------|--------|
| **Interested** → | Kalender-Link senden | Booked |
| **Question** → | Antworten + CTA | Follow-up |
| **Not now** → | 3-Monats-Follow-up | Nurture |
| **No fit** → | Archive | Closed |
| **Unsubscribe** → | Entfernen | Opt-out |

**Workflow:**
```
Reply kommt → Kategorisieren → Action → Status Update
```

**Daily Task:**
- [ ] Neue Replies checken (Instantly)
- [ ] Kategorisieren
- [ ] Antworten (innerhalb 4h!)
- [ ] CRM updaten (ClickUp oder Airtable)
- [ ] Meetings buchen

**Custom Fields:**
- Lead Name
- Campaign
- Reply Type
- Action Taken
- Status
- Meeting Date (falls gebucht)

## 🔗 Inbound + Outbound Synergie

### Content unterstützt Outbound:

1. **Social Proof**
   - YouTube Case Studies → In Cold Emails verlinken
   - LinkedIn Posts → In Signature

2. **Warmer Outreach**
   - Leads sehen Content vor Cold Email
   - "Saw your video about X" → Personalization

3. **Trust Building**
   - Content zeigt Expertise
   - Outbound konvertiert besser

### Outbound unterstützt Content:

1. **Content Ideen**
   - Fragen aus Cold Emails → YouTube Topics
   - Common Objections → LinkedIn Posts

2. **Distribution**
   - Cold Email List → Newsletter Subscriber
   - Reply: "Follow me on LinkedIn"
