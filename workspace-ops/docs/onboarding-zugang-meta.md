# 📘 Meta (Facebook) Ads — Zugang einrichten

## 1. Business Manager Zugang

1. Gehe zu **business.facebook.com** → Einstellungen (⚙️)
2. Links: **Partner** → **Hinzufügen**
3. Gib unsere Business Manager ID ein: **[ADSDROP_BM_ID]**
4. Berechtigungen aktivieren:
   - ✅ Werbekonten verwalten
   - ✅ Seiten verwalten
   - ✅ Pixel verwalten
5. **Anfrage senden**

## 2. Ad Account Zugang (Alternative)

Falls kein Business Manager vorhanden:

1. Gehe zu **facebook.com/adsmanager** → Einstellungen
2. **Kontozugang** → Personen hinzufügen
3. E-Mail eingeben: **deniz@adsdrop.de**
4. Rolle: **Werbetreibender** oder **Admin**
5. Bestätigen

## 3. Meta Pixel prüfen

1. Gehe zu **Events Manager** (business.facebook.com/events_manager)
2. Wähle deinen Pixel aus
3. Teile den Pixel-Zugang mit unserem Business Manager

## Was wir von dir brauchen:

| Info | Beispiel |
|------|---------|
| Business Manager ID | 123456789012345 |
| Ad Account ID | act_123456789 |
| Meta Pixel ID | 123456789012345 |

💡 **Tipp:** Die Business Manager ID findest du unter business.facebook.com → Einstellungen → Unternehmensinformationen.
