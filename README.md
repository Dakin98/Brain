# OpenClaw Workspace

Persönlicher OpenClaw Workspace für Deniz.

## Sync
- VPS und lokale Maschine synchronisieren via GitHub
- `.env` und Secrets sind lokal und werden nicht gepusht

## Struktur
- `skills/` — Custom Skills
- `memory/` — Tägliche Notizen
- `scripts/` — Hilfsskripte
- `AGENTS.md`, `SOUL.md`, `USER.md` — Konfiguration
