# Short-Form Mastery Guide

Der komplette Leitfaden für virale Reels, TikToks & Shorts.

---

## 1. Die Algorithmen verstehen

### Was alle Plattformen belohnen

**Completion Rate** — Wie viel % schauen bis zum Ende?
- 15s Video: 80%+ ist gut
- 30s Video: 60%+ ist gut
- 60s Video: 40%+ ist gut

**Engagement Rate** — Interaktionen pro View
- Shares > Saves > Comments > Likes
- Shares sind Gold (Algorithmus denkt: "Das will jeder sehen")

**Watch Time** — Gesamte Sekunden geschaut
- Mehr Views × längere Watch Time = Viralität

**Erste 3 Sekunden** — Entscheidend!
- 65% scrollen weiter, wenn der Hook nicht packt
- Dein Hook muss sofort den Pain Point treffen

## 2. Hook-Formeln (Copy & Paste)

### Die 15 besten Hooks

**1. Die Konfrontation**
```
"Stop! Mach das nicht mit deinen Meta Ads..."
[Pause, dann direkt in den Fehler]
```

**2. Die Zahl**
```
"3 Dinge, die jede 7-stellige E-Com Brand macht —
und du nicht."
```

**3. POV (Point of View)**
```
"POV: Du hast gerade herausgefunden,
warum dein ROAS sinkt..."
```

**4. Die Zeitersparnis**
```
"Das hat mir 2 Jahre gekostet zu lernen.
Ich zeig's dir in 30 Sekunden:"
```

**5. Der Fehler**
```
"Ich habe 50.000€ verbrannt mit diesem Fehler.
Vermeide ihn:"
```

**6. Das Geheimnis**
```
"Das erzählt dir keine Meta Ads Agentur:
[Geheimnis]"
```

**7. Die Transformation**
```
"Von 0 auf 100k Umsatz/Monat.
Ohne Influencer. Ohne viralen Content.
So geht's:"
```

**8. Die Checkliste**
```
"Wenn du diese 5 Dinge nicht checkst,
brennst du Geld mit Ads:"
```

**9. Die Frage**
```
"Warum scheitern 90% der E-Com Brands
bei der Skalierung?"
```

**10. Der Kontravers-Take**
```
"Kontravers: ROAS ist die irrelevanteste Metric.
Hier ist warum:"
```

**11. Das Ergebnis zuerst**
```
"Wir haben einen Client von 10k auf 1M
in 6 Monaten gebracht.
Das war unser Playbook:"
```

**12. Der Vergleich**
```
"Das hier ist ein 2x ROAS Ad.
Das hier ein 5x ROAS Ad.
Der Unterschied?"
```

**13. Die Story**
```
"Vor 1 Jahr war meine Agentur pleite.
Heute machen wir 7-stellig.
Der Wendepunkt:"
```

**14. Das Tutorial**
```
"So baust du in 60 Sekunden eine
Meta Ad, die konvertiert:"
```

**15. Der Mythos**
```
"Dieser Meta Ads 'Hack' zerstört deinen Account.
Hör auf damit:"
```

### Hook-Regeln
- Max. 8 Wörter für den ersten Satz
- Visuelle Pause nach dem Hook (1-2 Sekunden)
- Tonfall: überzeugt, nicht fragend
- Augenkontakt mit Kamera

## 3. Content-Struktur

### Das perfekte 30-Sekunden Script

```
[0-3s] HOOK — Stop the scroll
"Die meisten E-Com Brands verbrennen Geld mit Meta Ads."

[3-8s] PROBLEM — Relatable pain
"Sie optimieren auf ROAS. Das ist der größte Fehler."

[8-20s] LÖSUNG — Der Wert
"Was du stattdessen tracken solltest:
→ New Customer ROAS
→ CAC Payback Period
→ LTV/CAC Ratio"

[20-25s] BEWEIS — Warum glauben?
"Wir haben damit einen Client
von 200k auf 5M skaliert."

[25-30s] CTA — Was jetzt?
"Folgen für mehr Meta Ads Tips."
```

### Das perfekte 60-Sekunden Script

```
[0-3s] HOOK
[3-10s] PROBLEM (mehr Kontext)
[10-35s] LÖSUNG (3-5 Punkte, visuell unterstützt)
[35-50s] BEWEIS/BEISPIEL (Case Study)
[50-55s] ZUSAMMENFASSUNG
[55-60s] CTA
```

### Storytelling für Shorts

**Die 3-Act Structure (komprimiert):**
1. **Setup** (0-5s): Vorher, Problem
2. **Confrontation** (5-20s): Der Wendepunkt, Lösung
3. **Resolution** (20-30s): Nachher, Ergebnis, CTA

**PAS für Shorts:**
- Problem (3s)
- Agitate (5s) — Schmerz verstärken
- Solution (20s) — Die Lösung

## 4. Platform-Spezifika

### Instagram Reels

**Was funktioniert:**
- Educational + polished look
- Trending audio (aber nicht zu spät)
- Text overlays mit Branding
- 15-30s optimal
- 3-5 Hashtags in Caption

**Beste Posting-Zeiten:**
- Dienstag–Donnerstag: 11:00–13:00, 19:00–21:00
- Samstag: 9:00–11:00

**Caption-Struktur:**
```
[Hook als Text]

[1-2 Sätze Value]

[CTA]

#hashtag1 #hashtag2 #hashtag3
```

### TikTok

**Was funktioniert:**
- Raw, authentic, unpolished
- Trending sounds + effects
- Text-to-speech Voiceover
- Comment bait (Fragen stellen)
- 15-45s optimal

**Beste Posting-Zeiten:**
- Dienstag–Donnerstag: 7:00–9:00, 19:00–23:00
- Freitag: 19:00–23:00

**TikTok-spezifische Features:**
- Stitch (auf anderen Video antworten)
- Duet (Side-by-side)
- Green Screen (Reaktionen)
- Q&A (Community Fragen beantworten)

### YouTube Shorts

**Was funktioniert:**
- Educational content dominiert
- Clear value in Titel
- SEO-optimierte Beschreibung
- Cross-promote long-form Videos
- 30-60s optimal

**Beste Posting-Zeiten:**
- Täglich: 12:00–15:00, 19:00–22:00

**Titel-Formeln:**
- "Wie ich [Ergebnis] in [Zeitraum] erreicht habe"
- "Die Wahrheit über [Thema]"
- "X Dinge, die ich über [Thema] bereue"
- "Vermeide diesen [Thema] Fehler"

## 5. Content-Typen

### Educational (50% der Content)
**Ziel:** Wert liefern, Expertise zeigen

**Formate:**
- Quick Tips (1 Tipp, 15s)
- How-to Guides (Schritt-für-Schritt, 30-60s)
- Myth Busting (Fehlinfo korrigieren)
- Tool Recommendations
- Before/After Breakdowns

**Beispiel:**
```
Hook: "So testest du Creatives richtig"
Content: 
1. 3 Hooks testen
2. 3 Visuals testen  
3. Winning Combo skalieren
CTA: "Speichern für später"
```

### Storytelling (25% der Content)
**Ziel:** Persönliche Verbindung, Authentizität

**Formate:**
- Personal Journey
- Client Wins (anonymisiert)
- Failures & Lessons
- Behind the Scenes
- Day in the Life

**Beispiel:**
```
Hook: "Vor 2 Jahren war ich pleite"
Content:
- 3 Monate Cash runway
- 100 DMs pro Tag
- 2% Antwortrate
- Heute 7-stellig
CTA: "Folgen für die ganze Story"
```

### Contrarian (15% der Content)
**Ziel:** Diskussion anstoßen, Thought Leadership

**Formate:**
- Challenge industry norms
- Unpopular opinions
- "Das sagt dir keiner"
- Myth busting

**Beispiel:**
```
Hook: "ROAS ist irrelevant"
Content:
- Jeder redet drüber
- Aber es sagt nichts über Skalierung
- Was wirklich zählt: NC-ROAS, CAC Payback
CTA: "Stimmt ihr zu? Kommentar!"
```

### Trend-Jacking (10% der Content)
**Ziel:** Reichweite durch Trends

**Formate:**
- Trending audio + eigener Twist
- Popular formats adaptieren
- News/Updates kommentieren

**Beispiel:**
```
[Trending Sound]
Hook: "POV: Du merkst, dass dein
Meta Ads Account restricted ist"
[Reaktion auf Sound]
Content: Was tun?
CTA: "Folgen für mehr"
```

## 6. Produktion & Editing

### Equipment (Minimum)
- Smartphone (iPhone 12+ oder Android equivalent)
- Ringlicht oder natürliches Licht
- Mikrofon (optional, aber empfohlen)
- Stativ oder Gimbal

### Editing Apps
- **CapCut** (kostenlos, empfohlen)
- **InShot** (einfach)
- **Adobe Premiere Rush** (professionell)

### Editing-Best Practices
- **Jump Cuts** alle 2-3 Sekunden (Aufmerksamkeit halten)
- **Text Overlays** für Key Points
- **Zooms** auf wichtige Momente
- **Sound Effects** für Emphasis
- **Captions** immer (80% schauen ohne Ton)

### Thumbnails (YouTube Shorts)
- Gesichtsausdruck: überrascht/überzeugt
- Text: max. 3 Wörter
- Kontrast: hoch
- Branding: konsistent

## 7. Posting-Strategie

### Content-Kalender (Wochen-Plan)

**Montag:** Educational (Quick Win)
**Dienstag:** Storytelling (Personal/Client)
**Mittwoch:** Educational (How-to)
**Donnerstag:** Contrarian (Diskussion)
**Freitag:** Behind the Scenes (Agentur-Life)
**Samstag:** Trend-Jacking (wenn relevant)
**Sonntag:** Educational (Deep Dive)

### Posting-Frequenz
- **Minimum:** 1x pro Tag
- **Optimal:** 2-3x pro Tag
- **Power-User:** 3-5x pro Tag

**Wichtiger:** Konsistenz > Quantität

### Cross-Posting
1. Native auf jeder Plattform posten (nicht teilen!)
2. Leichte Anpassungen pro Plattform:
   - TikTok: rawer, mehr Text-to-speech
   - Reels: polished, mehr Visuals
   - Shorts: educational, SEO-optimiert

## 8. Analytics & Optimierung

### Was tracken?

**Primary Metrics:**
- Views (Reach)
- Completion Rate
- Shares (viral potential)

**Secondary Metrics:**
- Likes, Comments, Saves
- Profile Visits
- Link Clicks
- Follower Growth

### Was ist "gut"?

| Metric | Gut | Sehr gut | Viral |
|--------|-----|----------|-------|
| Views | 1k+ | 10k+ | 100k+ |
| Completion | 60% | 75% | 85%+ |
| Engagement | 5% | 10% | 15%+ |
| Shares | 1% | 3% | 5%+ |

### A/B Testing

**Was testen:**
- Hooks (erste 3 Sekunden)
- CTAs (letzte 5 Sekunden)
- Video Length (15s vs 30s vs 60s)
- Mit/ohne Face
- Text overlays vs clean

**Wie testen:**
- Gleiches Thema, verschiedene Hooks
- Poste beide Versionen (verschiedene Tage)
- Vergleiche nach 48h
- Winner wird Template

## 10. Don'ts

### Was du NICHT tun solltest

❌ **Langsame Intros** — Erste 3 Sekunden entscheiden
❌ **Zu viel Text** — Max. 5 Wörter pro Overlay
❌ **Kein CTA** — Verpasste Conversion-Chance
❌ **Copy-Paste von anderen** — Inspiriert sein = ok. Kopieren = nicht ok.
❌ **Inaktiv sein** — 1 Woche Pause = Algorithmus vergisst dich
❌ **Nur verkaufen** — 80% Wert, 20% Angebot
❌ **Falsche Aspect Ratio** — 9:16 immer (1080x1920)
❌ **Schlechte Audio** — Klarheit > Perfektion
